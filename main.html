Welcome to FizzBuzz

This program will take the number (x) you enter, and print out all the consecutive values from 1 to x.

If the current number is divisble by both 3 and 5, it will print FizzBuzz

If the number is divisible by 3, it will print Fizz

If the number is divisible by 5, it will print Buzz

Otherwise, it will just print the value
